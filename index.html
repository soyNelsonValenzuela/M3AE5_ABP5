Modelo de Caja

El modelo de caja es un concepto fundamental en CSS que describe cómo se calculan las dimensiones y el espaciado de los elementos en una página web. Cada elemento HTML se representa como una caja rectangular, y el modelo de caja define cómo se manejan sus márgenes, bordes, rellenos y contenido.
